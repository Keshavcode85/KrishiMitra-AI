# рҹҢҫ KrishiMitra AI

KrishiMitra AI is a smart agriculture web platform designed to help farmers with crop recommendations, plant disease detection, and an integrated agricultural marketplace.

This project combines Artificial Intelligence, weather data, and an e-commerce system to empower farmers with modern technology.

---

## рҹҡҖ Features

рҹҢұ **AI Plant Disease Detection**
- Upload plant leaf images
- AI predicts disease and suggests medicine

рҹҢҰ **Weather-based Crop Recommendation**
- Enter city name
- System fetches weather data
- Recommends suitable crops

рҹӣ’ **Agriculture Marketplace**
- Farmers can buy seeds, tools, fertilizers
- Add products to cart

рҹӨ– **Smart Agriculture Chatbot**
- Ask farming related questions
- Get instant suggestions

рҹ”җ **User Authentication**
- Register and login system
- Personalized cart experience
